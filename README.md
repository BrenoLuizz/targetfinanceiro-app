📱 Target — App de Metas Financeiras

O Target é um aplicativo mobile desenvolvido com React Native + Expo para ajudar usuários a criarem, acompanharem e gerenciarem metas financeiras de forma simples, moderna e intuitiva.

O app permite cadastrar objetivos financeiros, acompanhar a evolução de cada meta em tempo real e registrar movimentações de entrada e saída de dinheiro, funcionando com persistência local utilizando SQLite.

🚀 Funcionalidades

✅ Criar metas financeiras
Exemplo: viagem, celular, notebook, reserva de emergência.

📊 Acompanhar progresso da meta
Visualização do percentual alcançado e valor acumulado.

💰 Registrar transações financeiras

Entrada → guardar dinheiro
Saída → resgatar dinheiro

📋 Listar metas criadas
Exibe metas ordenadas por progresso.

🔎 Visualizar detalhes da meta
Mostra histórico completo de transações.

🗑️ Editar e remover metas
Com remoção automática das transações relacionadas.

🔁 Navegação dinâmica
Utilizando Expo Router baseado em arquivos.

💾 Persistência local offline
Dados armazenados diretamente no dispositivo com SQLite.

⚡ Atualização automática da interface
A UI reage instantaneamente às alterações do banco.
